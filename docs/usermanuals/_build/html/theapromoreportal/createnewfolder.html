Create new folderÔÉÅ

Folders help to organize the files for better management.

To create a folder, browse to the directory you want to create a folder and click on File -> Create folder.

Enter the folder name in the window displayed. Click OK.

The folder will be placed into the location/repository in which the user had initiated the plugin.
