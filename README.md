# Apromore Sphinx Documentation

* Install Sphinx
`sudo -H pip3 install -U sphinx`
* Check version
`sphinx-build --version`
* Browse to the desired directory and start sphinx
`sphinx-quickstart`
* Create the HTML documents from rst files
`make HTML`
